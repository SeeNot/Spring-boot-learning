# Reactive Library API

A learning project for java spring backend api using non-blocking technologies, related to a library  

## Tech Stack

* **Java 21**
* **Spring Boot 4.0.2**
* **Spring WebFlux**: For asynchronous, non-blocking web endpoints.
* **Spring Security**: Reactive JWT-based authentication and authorization.
* **Spring Data R2DBC**: Reactive relational database connectivity (PostgreSQL).
* **Docker & Docker Compose**: For seamless environment orchestration.
